# EventType

The values here will be prefixed with the event_category. For example, a customer created event will be `customer.created`


## Values

| Name                        | Value                       |
| --------------------------- | --------------------------- |
| `Created`                   | created                     |
| `Updated`                   | updated                     |
| `UpdatedStatusTransitioned` | updated.status_transitioned |
| `Deleted`                   | deleted                     |
| `Canceled`                  | canceled                    |
